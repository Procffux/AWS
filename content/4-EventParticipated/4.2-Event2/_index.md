### Speaker Sessions

#### 1. Bảo Huỳnh - Docker: A containerization technology

This session helped me understand Docker as a technology for packaging an application with its runtime environment, making deployment more consistent across local machines, servers, and CI/CD pipelines.

#### 2. Lê Hoàng Gia Đại - Combining AWS WAF with Machine Learning for Cyber Attack Detection

The speaker shared how AWS WAF can be combined with Machine Learning to detect abnormal traffic and improve web application protection. This helped me understand how cloud security can be enhanced through data analysis.

#### 3. Nguyễn Quốc Bảo - Multiplayer in the Cloud: Connecting Godot Clients with AWS WebSockets

This talk explained how to connect Godot game clients to a cloud backend through WebSocket. It gave me a clearer view of realtime applications and low-latency requirements in system design.

#### 4. Trường - Technical sharing session

This session allowed me to observe how a technical topic can be presented with context, structure, and practical examples. It was useful for improving my own report writing and workshop presentation skills.

#### 5. Việt Phát - AWS Neptune for Building a Graph Knowledge Base for GraphRAG

The speaker introduced Amazon Neptune and the idea of building a graph knowledge base for GraphRAG. I learned more about graph databases and how relationships between entities can support AI use cases.

#### 6. Vinh Trần - From IT Helpdesk to Senior Sysadmin

This career sharing session emphasized self-learning, practical experience, and system fundamentals. I learned that professional growth requires a clear roadmap, documentation habits, and continuous practice.

### Outcomes

Event 2 expanded my knowledge of Docker, web application security, WebSocket, GraphRAG, and system career development. I also learned how to capture key ideas and summarize technical content after a sharing session.
